## A simpler alternative to [VibranceGUI](https://vibrancegui.com/) for cs2 and Nvidia GPUs.

### To launch with Windows:
Put a shortcut in
```
%AppData%\Microsoft\Windows\Start Menu\Programs\Startup
```
